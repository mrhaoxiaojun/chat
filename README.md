Anonymous chat room
==============

> 匿名聊天室是简版


## 简单目录说明

>app.js为服务器端
>app/js/controller下面的为用angular定义的控制器
>app/js/directive下面为angular的指令
>app/pages下面为路由的页面
>app/vendor下面为用bower安装的插件
>front.js路由
>index.html首页
